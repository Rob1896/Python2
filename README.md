# Repositorio de prácticas en Python

Este repositorio contiene archivos de ejemplo y ejercicios para aprender diferentes conceptos de Python:

- **Variables**: cómo declarar y usar variables.
- **Funciones**: ejemplos de funciones simples y con parámetros.
- **Listas y diccionarios**: estructuras de datos básicas.
- **Programación Orientada a Objetos (POO)**: clases, objetos y métodos.

Cada archivo incluye ejemplos prácticos y explicaciones en los comentarios para facilitar el estudio.
